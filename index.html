  • 01 Solid Foundation

    기본기가 탄탄한 웹 퍼블리셔입니다.
    HTML, CSS, JavaScript의 기본기를 바탕으로, 시맨틱 마크업 과 웹 표준 을 준수하여 견고하고 유지 보수 가능한 웹사이트를 구축합니다.

  • 02 User-Focused

    사용자 경험을 최우선으로 고려합니다.
    ARIA 와 같은 접근성 기술을 활용하여 스크린 리더 지원을 강화하고, Grid & Flex 레이아웃과 미디어 쿼리를 활용하여 반응형 디자인 을 구현합니다.

  • 03 Meticulous

    세부 사항까지 신경 써서 작업합니다.
    작업을 진행할 때 작은 부분까지 신경을 쓰며, 실수를 최소화 하려 노력합니다. 안정적인 결과물을 만들어내는 것을 중요하게 생각합니다.

  • 04 Progressive

    지속적으로 발전하는 웹 퍼블리셔입니다.
    최신 웹 기술 에 관심을 가지고 활용하려고 하며, 지속적인 학습 을 통해 변화하는 트렌드에 맞추기 위해 노력합니다.
